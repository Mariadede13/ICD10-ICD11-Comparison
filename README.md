# ICD10-ICD11-Comparison

## Project Overview

This repository presents a comparison between the International Classification of Diseases, Tenth Revision (ICD-10), and the International Classification of Diseases, Eleventh Revision (ICD-11).

The project focuses on organising and comparing diagnostic codes and terminology in a structured format for educational and research purposes.

## Project Objectives

- Compare ICD-10 and ICD-11 diagnostic classifications.
- Document differences in diagnostic termonology.
- Build a structured dataset for further analysis.
- Develop Python scripts for data processing and visualization.

   ## Repository Contents

  - Data analysis and visualization (planned)
